# Text RPG Game
Простая текстовая RPG игра на С++, демонстрирующая основные принципы ООП
## Описание
Игрок может создать персонажа, выбрать его класс, зайти в инвентарь и использовать зелья.
Проект создан для демонстрации навыков в C++.

## Примененные знания и навыки
Наследование и полиморфизм в ООП:
- Базовый класс Character и классы-наследники Warrior, Mage, Archer.  
- Базовый класс Potion и подклассы HealthPotion, ManaPotion, StrengthPotion демонстрируют полиморфизм при использовании зелий.

Массивы:
- У персонажа есть инвентарь, в котором хранятся зелья.  
- Пользователь может использовать зелья из инвентаря, и они удаляются после использования.

Случайная генерация:  
- При создании персонажа ему случайным образом добавляется от 1 до 5 зелий в инвентарь.  
- Шансы выпадения каждого типа зелий примерно одинаковы
